# Net_consp_EJSP
Conspiracy Belief Systems:  A Cross-National Psychological Network Analysis of Conspiracism

## Description of the folder
The project follows the IPO protocol and is fully reproducible. 

Input -> Data and codebook
Processing -> Syntax replicating all analyses (Article and Supplement)
Output -> Material shown in the Article and Supplement, saved locally in high resolution

## Data availability
The data that support the findings of this study were collected as part of a project funded by the Italian Ministry of University and Research and are currently subject to a five-year embargo under the Ministry’s data dissemination policy. As such, the full dataset is not publicly available. However, upon publication, a reduced version of the dataset -including only the variables used in the present study- will be made available on Github. Please contact the corresponding author for any further data request.
