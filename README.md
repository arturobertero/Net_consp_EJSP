Replication material for the paper
"Conspiracy Belief Systems: A Multi-Study, Cross-National Psychological Network Analysis of Conspiracism"
---





##### Description of the folder

The project follows the IPO protocol and is fully reproducible.

Input → Data and codebooks

Processing → Syntax replicating all analyses (Article and Supplement)

Output → Materials shown in the Article and Supplement, saved locally in high resolution



##### Data availability

The full data for Study 1 were collected as part of a project funded by the Italian Ministry of University and Research and are currently subject to a five-year embargo under the Ministry’s data dissemination policy. Accordingly, the full Study 1 dataset is not publicly available. However, this repository includes a reduced version containing only the variables used in the present study.



The data for Study 2 and Study 3 are publicly available and must be downloaded separately to fully reproduce the analyses:



Study 2 data (Enders et al. 2023): https://doi.org/10.7910/DVN/MMMYGJ

&#x20;→ place in Input/enders/Qualtrics 2021.dta



Study 3 data (CCRS): https://doi.org/10.7910/DVN/VRRY9B

&#x20;→ place in Input/CCRS/CCRS.csv



All replication code is included in this repository. Once the data are placed in the correct folders, the scripts in the Processing directory will reproduce all results reported in the Article and Supplement.



For any additional data requests, please contact the corresponding author.
